# ☢️ Калькулятор Часу Безпечного Перебування в Зоні Радіоактивного Забруднення

Цей простий веб-додаток розроблено для швидкого розрахунку максимально допустимого часу перебування в зоні з іонізуючим випромінюванням за умови незмінної потужності дози та наявності захисту.

## 📌 Принцип роботи

Розрахунок базується на фундаментальній формулі дозиметрії, модифікованій для врахування захисних властивостей укриттів:

1.  **Фактична потужність дози ($\dot{D}_{факт}$)**:
    $$\dot{D}_{факт} = \frac{\dot{D}_{розвідки}}{К_{захисту}}$$
2.  **Час ($t$)**:
    $$t = \frac{D_{допустима}}{\dot{D}_{факт}}$$

Де:
* $D_{допустима}$ — Задана доза опромінення (мЗв).
* $\dot{D}_{розвідки}$ — Потужність дози випромінювання (мЗв/год) за даними розвідки.
* $К_{захисту}$ — Коефіцієнт ослаблення випромінювання захисним укриттям.

## 🚀 Використання

Додаток реалізовано в одному файлі (`index.html`) з використанням чистого HTML, CSS та JavaScript.

1.  **Клонуйте репозиторій**:
    ```bash
    git clone [посилання на ваш репозиторій]
    ```
2.  **Відкрийте файл**: Просто відкрийте файл `index.html` у будь-якому сучасному веб-браузері (Chrome, Firefox, Safari).
3.  **Мобільна адаптація**: Інтерфейс має збільшений шрифт та адаптивну верстку, оптимізовану для використання на смартфоні.

## ⚙️ Технічні деталі

* **HTML**: Структура калькулятора, поля введення та випадаючий список.
* **CSS**:
    * Адаптивний дизайн (`@media` та `viewport`).
    * Збільшений шрифт (`font-size: 18px` у `body`) для зручності на мобільних пристроях.
    * Кнопка **"РОЗРАХУВАТИ"** — зеленого кольору (`#4CAF50`).
    * Кнопка **"ОЧИСТИТИ"** — світло-сірого кольору (`#e0e0e0`).
* **JavaScript**: Обробка введення, виконання розрахунків та конвертація часу з годин у години й хвилини.

## 📋 Доступні коефіцієнти захисту ($К_{захисту}$)

| Значення Кз | Опис (Приклад) |
| :---: | :--- |
| **1** | Без захисту (відкрита місцевість) |
| **2** | Легкий захист (дерев'яний будинок, легкий одяг) |
| **4** | Автомобіль або легка будівля |
| **10** | Типова кам'яна будівля, верхні поверхи сховища |
| **50** | Спеціальне укриття, підвал з товстими стінами |
| **100** | Важке укриття, спеціалізований бункер |
